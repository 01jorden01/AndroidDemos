# AndroidDemos
Kleine Android demo's op bepaalde tehnologieen.

Behandelde materie:
- RXAndroid/RXJava
- Retrofit
- Android viewmodel
- Databinding
- AndroidX (opvolger SupportLibrary)


Vereisten:
- Android Studio 3.3 Canary 11 (of hoger)
- Android SDK platform-Tools 28.0.1
- API 28 Android 9.0 (Pie)
